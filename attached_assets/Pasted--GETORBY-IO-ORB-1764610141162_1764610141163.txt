SANDBOX SYSTEM (COPY FROM ORBIT)
--------------------------------------------------------------------------------
ORBIT has a complete Live/Sandbox mode system ready to implement. Core files:

1. ModeContext.tsx - Provider with useMode() hook
   - Tracks 'live' vs 'sandbox' mode
   - Persists to sessionStorage
   - Supports URL param ?mode=sandbox
   - enterSandbox(returnPath) / exitSandbox() functions

2. SandboxBanner.tsx - Visual indicators (4 variants)
   - SandboxBanner: Full-width cyan top bar
   - SandboxStatusCard: Glowing card with effects
   - SandboxStatusCompact: Small inline badge
   - SandboxBadge: Tiny "Demo" pill

3. SandboxWelcome.tsx - First-time welcome dialog
   - Shows on first sandbox entry
   - Role-based greeting
   - "Start Tour" or "Explore" options
   - Remembers if seen via sessionStorage

4. SandboxShiftCode.tsx - Go-live protection (OPTIONAL)
   - Daily rotating codes: ORBIT-XXXX
   - Supervisors can view/share codes
   - Users must enter code to exit sandbox
   - Set REQUIRE_SHIFT_CODE = true to enable

5. simulation.ts - Demo behaviors
   - Auto-response messages for actions
   - Page-specific contextual tips
   - Simulated notifications

6. useSandboxData.ts - Data switching hooks
   - useSandboxData(liveData, demoData)
   - useSandboxDataWithFallback(liveData, demoData, isLoading)
   - useSandboxAction(liveAction, sandboxAction)
